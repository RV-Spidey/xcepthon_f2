# Xcepthon

A React + Vite frontend application with Tailwind CSS and Framer Motion animations.

## Project Structure

- `src/` - React source files
  - `App.jsx` / `App.css` - Root component
  - `main.jsx` / `index.css` - Entry point and global styles
  - `components/` - Page section components (Hero, Navbar, About, FAQ, Schedule, Sponsors, Tracks, Footer)
- `public/` - Static assets including SVGs and parallax images
- `index.html` - HTML entry point

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite 5
- **Styling:** Tailwind CSS 3
- **Animations:** Framer Motion 12
- **Icons:** Lucide React

## Development

- Dev server runs on `0.0.0.0:5000`
- Start with: `npm run dev`

## Deployment

- Static deployment via `npm run build` → `dist/` directory
